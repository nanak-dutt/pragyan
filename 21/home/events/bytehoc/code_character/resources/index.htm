Problem Statement

You have an army of bots and towers. The main objective is to get as many of your troops into a specified flag area as possible. While the bots can move to any (permissible) location, the towers are stationary. You can transform bots into towers. You can also blast the bots and towers to lower the hp of all troops near it. But the trick here is, you are not playing the game, but you have to write an AI code to play the game.

 

FAQs:

1) Do I need to know C++ to participate?
Yes, a knowledge of C++ is definitely required. If you have an understanding of structs, arrays, pointers, and the STL library, you should be good to go.

2) Do I need to set up the environment on my local machine to code?
Not at all. You can compile and run your code directly in the integrated editor.

3) Is the event completely online?
Yes. Code Character is played completely on the web.
