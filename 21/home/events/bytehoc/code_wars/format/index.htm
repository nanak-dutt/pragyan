Format

Round 1: Strike 3

- It will be a coding contest on an online platform. There will be problems for which code will have to be written. Participants would be allowed only three submissions per problem, after which the problem will be locked.

- It is an eliminative round.

- Platform: Codechef

 

Round 2: Code Golf

- The top 32 people will be selected from Round 1 for this round.

- In this round, participants will have to solve the problem on an online platform. The given set of problems has to be solved using a minimum number of characters. (Characters mean alphabets, numbers, or symbols).

- Platform: Hackerank

 

Round 3: Code Battle

- The top 16 individuals will be selected from Round 2 for this round

- In this round, there will be a 1v1 knockout battle conducted on an online platform. Each battle will be eliminative, and the person who is the quickest and solves the problem correctly is the winner of that battle and can proceed to the next one.
